riftapi — local test page

Click any endpoint to call the live API at http://127.0.0.1:18080
Base URL:
GET /health Sync health: card count, last sync timestamp, last error. 






  

    GET
    /sets
    List all sets (paginated).
  

  

    
    
  

  






  

    GET
    /sets/set-id/{set_id}
    Get a single set by its code (e.g. OGN).
  

  

    
    
  

  






  

    GET
    /cards/{riftbound_id}
    Get a single card by its riftbound id. The card preview (image + stats) is shown below the JSON.
  

  

    
    
  

  






  

    GET
    /cards
    List cards with filters. set_id optional; size clamps to [1, 100].
  

  

    
    
    
    
  

  






  

    GET
    /cards/search
    Search by name (and rules text). Matches the bot's /card command's primary path.
  

  

    
    
    
  

  






  

    GET
    /cards/random
    One card chosen uniformly at random from the local store. The card preview is shown below the JSON.
  

  

    
  

  






  

    GET
    /cards/riftbound/{riftbound_id}
    All prints for a given riftbound id. Bare list (not paginated). The first card preview is shown.
  

  

    
    
  

  






  

    GET
    /index/card-names
    All card names sorted alphabetically. Used by the bot for the names index.
  

  

    
  

  






  

    GET
    /index/domains
    All unique domain values across the card set.
  

  

    
  

  






  

    GET
    /index/types
    All unique card-type values.
  

  

    
  

  






  

    GET
    /index/rarities
    All unique rarity values.
